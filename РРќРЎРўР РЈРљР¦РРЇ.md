# 🐺 White Wolf Bot — Инструкция запуска

## Шаг 1 — Заполни .env файл

Открой файл `.env` и замени все значения:

```
BOT_TOKEN=        ← новый токен от @BotFather (/revoke → получи новый)
CHANNEL_ID=       ← @whitewolf_fx (username твоего канала)
ANTHROPIC_API_KEY=← с сайта console.anthropic.com → API Keys
ADMIN_ID=         ← твой Telegram ID (узнай у @userinfobot)
```

## Шаг 2 — Установи зависимости

Открой папку в PowerShell и выполни:

```
pip install -r requirements.txt
```

## Шаг 3 — Добавь бота в канал

1. Зайди в канал → Управление → Администраторы
2. Добавь своего бота
3. Включи право: Публикация сообщений ✅

## Шаг 4 — Тест

```
python test_post.py
```

Проверь что пост пришёл в канал. Если всё ок — деплой.

## Шаг 5 — Деплой на Railway (бесплатно, 24/7)

1. Зарегистрируйся на railway.app
2. New Project → Deploy from GitHub (загрузи папку)
3. Variables → добавь все 4 переменные из .env
4. Deploy → готово!

## Расписание постов

- 08:00 — Утренний обзор рынка (Пн-Пт)
- 09:00 — Обзор выходного дня (Сб)
- 13:00 — Погода рынка / новости (Пн-Пт)
- 20:00 — Итоги дня (каждый день)
- 20:30 — Мотивация (только Пятница)
